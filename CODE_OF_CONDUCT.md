# Código de conducta

## Nuestro compromiso

Este proyecto quiere ser un espacio acogedor y respetuoso para todo el mundo,
sin importar edad, identidad o expresión de género, discapacidad, etnia, nivel
de experiencia, nacionalidad, aspecto, religión u orientación sexual.

## Qué se espera

- Trata a los demás con respeto y empatía.
- Acepta las críticas constructivas con buena disposición.
- Céntrate en lo que es mejor para la comunidad y el proyecto.
- Usa un lenguaje acogedor e inclusivo.

## Qué no se tolera

- Lenguaje o imágenes sexualizadas, insultos o ataques personales.
- Acoso público o privado de cualquier tipo.
- Publicar información privada de terceros sin permiso.
- Cualquier conducta que razonablemente se considere inapropiada.

## Aplicación

Si presencias o sufres una conducta inaceptable, repórtala en privado a través
del **aviso de seguridad de GitHub** del repositorio
([Security → _Report a vulnerability_](https://github.com/Mun1to/Vocript/security/advisories/new))
o abriendo una _issue_ si no es sensible. Las quejas se revisarán y se actuará
según se considere necesario, incluyendo avisos o el bloqueo de quien incumpla
estas normas.
